% 11-29：羔羊的血与肉的比喻

### 羔羊

本文经文 启 7:14 羔羊的血 → 洗净衣服

- 起源：出 12:1-14 逾越节宰杀羔羊
- 成就：来 9:9-10 律法 → 影子
- 真意：约 1:29 耶稣

### 羔羊的血和肉

- 起源：利 17:11 血 → 生命 → 赎罪
- 成就：约 1:1-4 道 → 生命
- 实体：
    - 约一 1:1-2 生命之道 = 耶稣
    - 血与肉：耶稣的生命的话语
        - 初临
            - 约 6:53-57 吃耶稣的血和肉 → 永生
            - 约 6:68 彼得：只归从永生之道
            - 约 5:24-25 出死入生
        - 再临
            - 启 7:14 耶稣的血 → 洗净衣服
            - 启 22:14 洗净衣服 → 圣城

### 逾越节

- 起源：出死入生
- 初临：旧启（受印 + 传讲）出死入生
- 再临：新启（受印 + 传讲）出死入生
