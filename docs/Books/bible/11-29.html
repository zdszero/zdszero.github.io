羔羊的血和肉

  • 起源：利 17:11 血 → 生命 → 赎罪
  • 成就：约 1:1-4 道 → 生命
  • 实体：
    • 约一 1:1-2 生命之道 = 耶稣
    • 血与肉：耶稣的生命的话语
      • 初临
        • 约 6:53-57 吃耶稣的血和肉 → 永生
        • 约 6:68 彼得：只归从永生之道
        • 约 5:24-25 出死入生
      • 再临
        • 启 7:14 耶稣的血 → 洗净衣服
        • 启 22:14 洗净衣服 → 圣城
